# CruiseLink SaaS — 일본 여행사 대상 제안서

**크루즈링크(CruiseLink)** | 2026년 4월

---

## 1. 제안 개요

크루즈링크는 글로벌 9개 크루즈 선사의 실시간 상품·재고·요금 데이터를 통합 제공하는 **크루즈 특화 B2B SaaS 플랫폼**입니다.

일본 내 크루즈 전문 여행사·GSA를 대상으로, **맞춤형 화이트라벨 웹서비스**를 제공합니다. 여행사는 자사 브랜드로 크루즈 예약 서비스를 즉시 운영할 수 있으며, 선사별 API 연동·상품 데이터 관리·예약 시스템을 크루즈링크가 일괄 담당합니다.

---

## 2. 현재 일본 시장의 과제

| 과제 | 상세 |
|------|------|
| **선사별 개별 대응** | MSC·Princess·HAL·NCL 등 각 선사의 예약 시스템이 상이하여, 여행사 담당자가 선사마다 별도 교육·운영 필요 |
| **수동 견적·예약** | 재고 확인→견적 작성→예약 확정까지 평균 2~3일 소요. 인건비 부담 |
| **웹서비스 구축 비용** | 자체 개발 시 초기 3,000만엔~5,000만엔, 유지보수 월 100만엔~300만엔 |
| **실시간 데이터 부재** | 선사 프로모션·요금 변경·잔여 객실 정보가 지연 반영되어 기회 손실 |
| **다국어 대응 한계** | 일본어 고객 응대에도 인력 집중, 영문 선사 자료 해독에 추가 리소스 |

## 3. 크루즈링크 SaaS 제공 서비스

### 3.1 화이트라벨 크루즈 예약 웹서비스

```
┌─────────────────────────────────────────────┐
│         [여행사명] 크루즈 예약 사이트          │
│     https://cruise.*****-travel.co.jp        │
│                                              │
│  ┌ 검색 ──┐ ┌ 상품상세 ──┐ ┌ 예약 ──┐      │
│  │ 출발일  │ │ 일정표     │ │ 견적서  │      │
│  │ 목적지  │ │ 선박정보   │ │ 예약확정 │      │
│  │ 선사   │ │ 요금비교   │ │ 바우처  │      │
│  └────────┘ └───────────┘ └─────────┘      │
│                                              │
│  ※ 크루즈링크 엔진이 백엔드에서 자동 구동       │
│  ※ 여행사 브랜드·도메인·디자인 커스텀 가능      │
└─────────────────────────────────────────────┘
```

- **여행사 자체 도메인**에서 운영 (예: cruise.travel-xxx.co.jp)
- **여행사 브랜드** 노출 (로고, 컬러, 연락처)
- 크루즈링크 로고는 표시하지 않거나, "Powered by CruiseLink"로 최소 표기

### 3.2 통합 크루즈 데이터 API

| 항목 | 내용 |
|------|------|
| **대상 선사** | MSC, Royal Caribbean, Carnival, Princess, NCL, Holland America, Celebrity, Costa, Disney Cruise (9개사) |
| **데이터** | 실시간 일정·요금·잔여객실·프로모션 |
| **갱신 주기** | 일 1회 자동 업데이트 (프로모션은 실시간) |
| **데이터 건수** | 21,000개 이상의 크루즈 상품 |
| **일본어 지원** | 상품명·기항지·선박정보 일본어 자동 변환 |

### 3.3 AI 챗봇 (크링AI)

- 고객 24시간 크루즈 상담 자동 응대
- 일본어 자연어 질문 → 최적 상품 추천
- 여행사 영업시간 외 문의에도 자동 응대하여 리드 손실 방지

### 3.4 관리 백오피스

- 예약 현황 대시보드
- 고객 관리 (CRM)
- 매출 리포트
- 프로모션 등록·관리

## 6. 도입 절차

```
[1] 상담·요구사항 정의 (1~2주)
    ↓
[2] 계약 체결
    ↓
[3] 화이트라벨 사이트 구축 (2~4주)
    ├─ 도메인 설정
    ├─ 브랜드 디자인 적용
    ├─ 크루즈 데이터 연동
    └─ 일본어 로컬라이제이션
    ↓
[4] 테스트·검수 (1~2주)
    ↓
[5] 정식 오픈
    ↓
[6] 운영·유지보수 (지속)
```
